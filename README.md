# API-for-a-blog-using-Express-and-MySQL-

Create an API using Express and MySQL for a blog.

1. Api to save new blog post with fields (title, description, author)
2. Update each post using post id.
3. Get all posts.
4. Get all posts by author id.
5. Delete post by id.
6. On saving each post, a mail notification about the post should be sent to all other authors.
